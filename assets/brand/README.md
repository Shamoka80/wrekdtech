# Wrek'd Tech Brand Assets

This folder stores the website logo assets used for header, contact-page, and footer branding.

## Current asset map

| File | Intended section | Source selection |
| --- | --- | --- |
| `wrekd-tech-header-logo.svg` | Header logo | Derived from `FULL LOGO 2.png` |

| `wrekd-tech-footer-mark.svg` | Footer compact mark | Derived from `Logo Symbol.png`; preferred over `mini.logo.png` because it preserves the same mark at cleaner resolution |

## Implementation rules

- Keep logo backgrounds transparent.
- Do not stretch logo files; use `height: auto` and `object-fit: contain`.
- Header: use the full horizontal lockup at approximately 230–260px wide on desktop.
- Contact page: use the balanced full lockup centered above the Contact label at approximately 180–220px wide.
- Footer: use the compact mark at approximately 44–56px wide/tall.

These assets were prepared from the real Drive logo imagery rather than text-only wordmark files.
