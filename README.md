# Wrek'd Tech — Landing Page

Public static landing page for **Wrek'd Tech, LLC**.

**Tagline:** Greener Thinking, Smarter Recycling

Live site target: <https://wrekdtech.com>

## Purpose

This repository contains the first public GitHub Pages landing page for Wrek'd Tech. The page is a public credibility anchor for exploratory academic, agency, incubator, advisor, and technical collaboration conversations.

It is not a recycling service site, repair service site, customer portal, franchise page, investor page, or the R2 Ready application.

Deploy through GitHub Pages from the `main` branch root.

```text

```

After the files are committed, configure GitHub Pages in repository settings and then configure the domain DNS records through the domain registrar.
